# Zoowarnについて

このプログラムは事情がありZoomを退出する余裕がない時に活用できるプログラムです。

設定した人数以下になると自動的に退出します。


## 注意点

Zoomはミーティングコントロールが常に表示されるようにしておいてください。

(ZoomではAltキーを押すと表示・非表示が切り替えられます)
